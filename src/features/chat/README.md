# Chat Feature

This feature module contains Chat-related UI and logic.

## Structure

- **components/**: Presentational and container components for chat
  - `ChatPanel`: Main chat panel component for AI interactions
- **hooks/**: View-model hooks for chat agents and workflows
- **state/**: Store slices/selectors for chat state (planned Stage 2)

## Components

All chat components have been migrated from `src/components/` to this feature module following the feature-first architecture pattern.
