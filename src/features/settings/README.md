# Settings Feature

This feature module contains Settings-related UI and logic.

## Structure

- **components/**: Settings panels and subcomponents
  - `SettingsPanel`: Main settings panel component
- **hooks/**: View-model hooks for settings behaviors
- **state/**: Store slices/selectors for settings (planned Stage 2)

## Components

All settings components have been migrated from `src/components/` to this feature module following the feature-first architecture pattern.
